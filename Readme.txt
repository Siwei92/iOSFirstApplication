This sudoku game has three player modes(easy,medium and difficult) can be choosen by users. It also has a test mode which is for developer.

When player click the grid the game will suggest possible hint for users by change grid colors.

There is a timer in the game to record score.

Player can erase the number which has filled

Player can reset the grid to play the game again.

The game contain three kinds of audio when meet error click and success situations

The game has been connected to database to preserve history data.

The game has a activity to present the best time list which is sorted by the time comsumptions of users.
